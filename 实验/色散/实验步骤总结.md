# 色散与偏振实验 —— 详细操作步骤

本实验共 **四大部分**，需在课堂内（截止17:30）完成。以下按实际操作顺序整理。

---

## 第一部分：光的色散与三棱镜折射率测量（必做）

### 实验目标
- 搭建分离式平台分光光谱仪
- 用**最小偏向角法**测定三棱镜折射率（测 **2 条谱线**，建议黄光587.6nm和绿光504.8nm）
- 计算其中一条谱线的折射率不确定度 Δn

### 详细步骤

#### 步骤 1：等高共轴粗调
1. 将三棱镜放在平台上，用机械夹具卡紧（防止滑落）
2. 望远镜焦距调到无穷远（对远处聚焦，或目镜筒向左旋到最大），放在转台夹具上
3. 按顺序摆放器件：**氦灯 → 狭缝 → 凸透镜 → 转台 → 望远镜**
4. 目测调整所有器件达到等高状态
5. 用卷尺辅助粗调共轴

#### 步骤 2：调焦（产生平行光）
1. 氦灯靠近狭缝，使透过狭缝的光最亮
2. 凸透镜焦距 f=20cm，用卷尺量出大致距离放好
3. 在望远镜中观察狭缝，调节狭缝宽窄螺钉（**不要过细**）
4. 改变凸透镜距离，直到望远镜中看到**清晰无视差**的狭缝像
5. 调整十字划线处于水平竖直

#### 步骤 3：寻找氦光谱
1. 将三棱镜放在转台上（注意放置方向，参考讲义图1-9）
2. **先用肉眼**在出射侧找到凸透镜框的像，小幅度转动转台寻找氦光谱
3. 可借助激光笔找出射光大致位置（**找到后务必关闭激光笔，不要直视！**）
4. 肉眼看到后，把望远镜旋到该位置，在望远镜中找到清晰的氦光谱

#### 步骤 4：测量最小偏向角
1. 观察到谱线后，向一个方向**缓慢**转动转台，使谱线偏向角减小
2. 当谱线即将反向运动时 → **该临界位置就是最小偏向角**
3. **锁死转台锁紧螺钉**
4. 转动望远镜，使分划板竖直线与狭缝像对准
5. 记录该状态转盘角度值 **φ**（估读一位）
6. **换另一条谱线**，重复上述操作，记录角度 φ'

#### 步骤 5：记录入射角
1. **取下三棱镜**
2. 转动望远镜使分划板竖直线与狭缝像对准
3. 记录入射方位角度值 **φ₀**

#### 步骤 6：数据处理（课堂完成）
1. 最小偏向角 δ = φ - φ₀（出射角 - 入射角）
2. 折射率公式：**n = sin[(A+δ)/2] / sin(A/2)**，其中 A = 60°（等边三棱镜）
3. 计算 2 条谱线的折射率 n
4. 计算其中一条谱线的不确定度 Δn（**角度需要转换为弧度**）

> [!IMPORTANT]
> 氦放电管高压 5~6kV，**严禁**用手碰电极！光学元件不要用手触摸光学面！

## 第二部分：虹和霓（必做）

### 实验目标
- 用布儒斯特角测定三棱镜折射率并校正偏振片方位
- 观察玻璃球的虹和霓，验证其偏振特性
- 用激光观察亚克力棒的虹和霓

### 详细步骤

#### 实验 2-1：布儒斯特角测定折射率 + 校正偏振片
1. 三棱镜放在转台上，用夹具固定
2. 半导体激光笔放到夹具上，调整高度和角度，使玻璃**反射光与激光笔出射重合**（此时激光方向即为法线）
3. 偏振片放在激光笔和转台之间
4. 让激光穿过偏振片打到三棱镜光学面，反射光投射到大白屏上
5. 预设入射角 **θᵢ ≈ 57°**
6. 旋转偏振片到某一位置使反射光点最暗
7. 反复微调入射角 θᵢ 和偏振片方位角，使反射光点**最暗**
8. 读取入射角，即布鲁斯特角 θB，折射率 **n = tan(θB)**
9. 记录好两个偏振片校正后的方位角度

#### 实验 2-2：白光观察虹和霓
1. 先后将**大玻璃球和小玻璃球**放在转台上
2. 手电筒放在夹具上，调整焦距和光照强度
3. 照射玻璃球，**观察虹（一次反射）和霓（二次反射）**
4. 用步骤 2-1 中校正好的偏振片旋转观察，判断虹和霓的偏振状态
   - 理论：虹和霓中 **S 光占主导**，偏振方向沿彩虹圆环切线方向

#### 实验 2-3：激光观察虹和霓
1. 实验中有 1~15mm 不同直径的亚克力棒，选择合适的
2. 用双面胶将亚克力棒粘在转台上（粗棒可不用）
3. 激光笔放在夹具上，照射亚克力棒
4. 观察**幻日、虹和霓**现象
5. 测定虹和霓的出射角度，计算亚克力的折射率

#### 选做：精确测定亚克力棒折射率
1. 用游标卡尺测出准确直径
2. 借助实验盒器件自行设计测量方法
3. 测定入射角和出射角，计算折射率

## 第三部分：特斯拉线圈（必做）

### 实验目标
- 观察特斯拉线圈的放电现象
- 理解无线发光和电弧放电的原理

### 详细步骤
1. **观察基本结构**：了解特斯拉线圈的构造（LC振荡回路、升压线圈）
2. **通电观察放电现象**：观察电弧放电（类似小型闪电）
3. **无线点亮实验**：
   - 分别用特斯拉线圈点亮 **LED灯泡、节能灯管、8种气体**
   - 分析三种无线发光的不同机理
4. **播放音乐**：利用特斯拉线圈播放一段音乐，分析为什么能播放音乐

> [!CAUTION]
> - 特斯拉线圈**远离**示波器、信号发生器
> - **手机、手环等电子产品远离**特斯拉线圈（拍照时尤其注意，可能被电磁感应烧坏！）
> - 手持灯泡时**不要距离过近**，防止电弧打到
> - 放电会电离空气产生臭氧，**实验室要通风**，观察完尽快关闭线圈

## 第四部分：光谱仪实验（必做）

### 实验目标
- 学习使用光栅光谱仪测量和分析不同光源的光谱
- 对比测量值与标准值，分析误差来源

### 详细步骤

#### 步骤 1：实验准备
1. 光谱仪用 USB 线连接电脑，放在转台上固定好
2. 电脑打开桌面 "窗台小圃光谱仪软件和资料" 文件夹
3. 打开 **"Theremino_Spectrometer_窗台小圃_通用版.exe"** 启动软件

#### 步骤 2：添加光源
1. 将氦管（或荧光灯）连接电路
2. 对准光谱仪的通光孔，让光尽可能多地进入
3. 注意调节好**高度和角度**，只有直射光才能产生较明显信号

#### 步骤 3：调制谱线
1. 相机中出现明显光谱信号后，调节**高度和起始位**的数值
2. 让选区框的高度略低于各个峰高（光谱信号覆盖选区框）
3. 起始位置和结束位置调节选区框宽度（可设默认值测全波段）

#### 步骤 4：调节曝光
1. 点击软件 **"视频控制"**
2. 取消勾选 **"自动"** 方框
3. **手动调整**曝光强度，使主峰尽可能尖锐，同时滤去杂光谱项

#### 步骤 5：波长校准（只需一次）
1. 依次点击 **【工具】→【校准波长】→【手动输入校准波长】**
2. 输入两个校准波长（氦光建议选 **447.1nm** 和 **706.6nm**）
3. 在光谱图中找到对应的峰
4. **拖动参考线对准**（关键：找准对应的峰！）

#### 步骤 6：光谱测量与分析
1. 校准完成后，记录其他峰的波长值
2. **换用其他光源**（氢管、荧光灯等），重复操作，记录光谱图
3. 对比标准光谱图，定性分析强度分布
4. 定量分析特征峰波长分布
5. 比较标准值与测量值的差异，分析误差来源
6. 思考改进方法
