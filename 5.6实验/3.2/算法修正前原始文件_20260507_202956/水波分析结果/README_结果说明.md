# 3.2 / 4.mp4 水波测量说明

## 结论使用优先级

本文件夹中的 x-t 图和方向分离结果只作为辅助诊断，不建议直接作为最终实验结论。

原因：

- 本视频只有 117 帧，时长约 3.90 s，时间窗较短。
- x-t 摘要里的 `stable_window_std_cm` 接近 0，主要是频域网格锁定造成的假稳定，不代表真实波长完全恒定。
- 主波段方向分离给出的 `incident_fft_lambda_cm = 20.22 cm` 与直接主波包测量 `24.42 ± 0.41 cm` 差异较大，说明短片条件下主峰/频带选择不够稳。

## 可引用数据

- 局部相邻小波峰：`7.81 ± 0.80 cm`，见 `4_wave_crest_summary.csv`。
- 直接主波包峰距：`24.42 ± 0.41 cm`，见 `4_wave_crest_summary.csv` 和 `4_crest_distances.csv`。

## 建议写法

本组视频时长较短，x-t 图可用于观察传播方向和反射/叠加风险，但不单独给出最终波长。最终波长应优先依据清晰可辨的波峰直接测量，并在报告中说明 x-t 结果仅用于辅助判断。
