## 必背对比表

| 类型 | 写法 | 有序 | 可修改 | 可重复 | 常见用途 |
|---|---|---:|---:|---:|---|
| list | `[1, 2, 3]` | 是 | 是 | 是 | 存一组可变数据 |
| tuple | `(1, 2, 3)` | 是 | 否 | 是 | 存固定不变的数据 |
| dict | `{"name": "Tom"}` | 按键访问 | 是 | 键不可重复 | 存属性、映射关系 |
| set | `{1, 2, 3}` | 否 | 是 | 否 | 去重、交并差 |

特别注意：

- 空 dict：`{}`
- 空 set：`set()`
- 单元素 tuple：`(5,)`
- list 用索引访问：`a[0]`
- dict 用键访问：`student["name"]`
- set 无序，不能靠下标取第几个。

## 文件 I/O 速记

打开模式：

- `r`：只读，文件必须存在。
- `w`：只写，会清空原文件，不存在则创建。
- `a`：追加写，不清空原文件。
- `rb` / `wb`：二进制读写。

推荐写法：

```python
with open("data.txt", "r", encoding="utf-8") as f:
    content = f.read()

with open("out.txt", "w", encoding="utf-8") as f:
    f.write("hello\n")
```

逐行读取：

```python
with open("data.txt", "r", encoding="utf-8") as f:
    for line in f:
        print(line, end="")
```

## 异常处理速记

```python
try:
    a = int(input("a: "))
    b = int(input("b: "))
    print(a / b)
except ZeroDivisionError:
    print("除数不能为 0")
except ValueError:
    print("请输入数字")
else:
    print("没有异常时执行")
finally:
    print("无论是否异常都会执行")
```

要点：

- try 放可能出错的代码。
- except 捕获错误。
- else 在没有异常时执行。
- finally 一定执行。

## OOP 速记

```python
class Student:
    def __init__(self, name, score):
        self.name = name
        self.score = score

    def show(self):
        print(self.name, self.score)

s = Student("Tom", 90)
s.show()
```

继承：

```python
class Person:
    def __init__(self, name):
        self.name = name

class Student(Person):
    pass

s = Student("Alice")
print(s.name)
```

要点：

- class 是类，对象是类创建出来的实例。
- `__init__` 是初始化方法。
- self 代表当前对象。
- 继承用于复用父类代码。
