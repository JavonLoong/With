复习总原则

所有计算题统一按这条链路启动：

受力图 -> 截面法 -> 平衡方程 -> 内力图/内力函数 -> 应力或变形 -> 强度/刚度/稳定/疲劳校核

考前检查清单

  • 是否先画受力图并标清外力、约束、尺寸、截面位置？
  • 是否用截面法求出了 N,V,M,T，而不是把外载直接塞进公式？
  • 是否统一单位：N-mm-MPa 或 N-m-Pa？
  • 是否区分强度、刚度、稳定、疲劳寿命？
  • 是否找到了危险截面和危险点？
  • 是否明确写出校核式，例如 sigma_max <= [sigma]、delta_max <= [delta]、P <= Pcr/n？
  • 是否说明最终结论：安全/不安全，控制因素是什么？

标准答题格式

  1. 画受力图。
  2. 求支反力。
  3. 取截面并列平衡。
  4. 画或写内力图/内力函数。
  5. 计算应力、应变、位移、转角或临界载荷。
  6. 选择强度、刚度、稳定或疲劳准则。
  7. 写清结论和单位。

第01章 材料力学导论

1. 章节一句话总纲

材料力学研究杆件在外力作用下的内力、应力、变形与失效风险，解题总流程是：受力图 -> 截面法 -> 平衡 -> 内力 -> 应力/变形 -> 强度、刚度或稳定校核。

2. 核心问题

问题考试抓手
材料力学研究什么？杆件、内力、应力、应变、强度、刚度、稳定性
为什么引入应力？内力大小还不够，要看单位面积上的受力
强度、刚度、稳定性区别？强度防破坏，刚度限变形，稳定防失稳

第09章 压杆的稳定性分析与稳定性设计

1. 章节一句话总纲

受压细长杆可能在压应力未达到强度极限前发生侧向失稳，所以压杆必须用临界力、柔度和稳定安全系数校核。

2. 核心问题

问题考试抓手
稳定失效与强度失效区别？不是材料压坏，而是整体弯曲失稳
欧拉临界力是什么？Pcr=pi^2 EI/(mu L)^2
端部约束如何影响稳定？通过长度系数 mu
弱轴为什么重要？稳定绕 Imin 方向发生
设计怎么判断？同时满足强度和稳定
欧拉公式什么时候能用？柔度足够大、临界应力未超过比例极限时使用

3. 前置知识

  • 轴向压缩 sigma=P/A。
  • 梁弯曲刚度 EI。
  • 惯性矩 I、面积 A、惯性半径 i=sqrt(I/A)。

4. 概念依赖图

轴向压力 -> 受力图 -> 端部约束 -> mu L -> Imin/A/i -> 柔度 lambda -> Pcr -> 稳定校核

5. 公式血缘表

公式来源适用条件常见误用
Pcr=pi^2 EI/(mu L)^2欧拉稳定理论细长压杆、线弹性忘记有效长度
sigma_cr=Pcr/A=pi^2 E/lambda^2欧拉公式除以面积细长杆用于短粗杆
lambda=mu L/i柔度定义判断长细程度把 i 写成 I/A
i=sqrt(I/A)截面几何任意截面不取弱轴
[P]st=Pcr/nst稳定安全系数设计校核把 Pcr 当许用载荷

公式血缘审计补充

欧拉公式前提是细长压杆、轴向受压、线弹性、初始缺陷不作为主控量；不适用于短粗柱直接压溃、中等柔度需经验公式、偏心受压未处理弯曲的题。mu,L,Imin,A,i,lambda 必须取同一失稳平面；Pcr 为临界力，设计要除以稳定安全系数；典型题入口是端部约束识别、弱轴稳定校核和强度/稳定双校核。

欧拉适用范围

考试中可用这个判据：先算 lambda=mu L/i，再算 sigma_cr=pi^2 E/lambda^2。若 sigma_cr 不超过材料比例极限 sigma_p，或题目明确说明“细长压杆/适用欧拉公式”，可用欧拉公式；否则属于中小柔度杆或短柱，应改用题给经验公式或先按强度压缩校核。没有给 sigma_p 时，期末题通常会用“细长杆”提示。

6. 典型题型表

题型标准流程

安全系数nst=Pcr/P
许用压力[P]st=Pcr/nst

综合校核先强度 P/A <= [sigma]，再稳定 P <= Pcr/nst

  • 没有检查 sigma_cr 是否仍在线弹性比例范围。
  • 临界力没有除以稳定安全系数。

第12章 动载荷与动应力概述

1. 章节一句话总纲

动载荷题把速度、加速度、冲击和突加载荷转化为等效内力或能量关系，再求最大动应力并进行强度校核。

  • 静应力公式。
  • 轴向变形和梁挠度。
  • 动能、势能、功。

4. 概念依赖图

动载类型 -> 受力图/能量图 -> 等效动载或动载系数 -> 动内力 -> 动应力 -> 校核

5. 公式血缘表

公式来源适用条件常见误用
Kd=sigma_d/sigma_st=P_d/P动载放大定义线弹性、同受力形式与安全系数混淆
Kd=2突加载荷能量法瞬时从 0 加到 P当缓慢加载用
Kd=1+sqrt(1+2h/delta_st)重物冲击能量守恒线弹性冲击delta_st 取错

公式血缘审计补充

动载公式的前提是可等效为单自由度线弹性响应、阻尼忽略或按题意忽略；不适用于超过屈服后的冲击、复杂振动或多自由度耦合题直接套系数。delta_st 必须是同一载荷作为静载时的位移；h,delta_st 单位一致；典型题入口是突加载荷、重物冲击、加速吊索和动强度校核。

6. 典型题型表

动强度校核求最大动应力 -> 比 [sigma]

  • 惯性力方向取错。
  • 只算应力不做校核。

8. 最低掌握标准

  • 会用 Kd=2 和冲击系数。
  • 会处理简单加速吊索。

9. 高分掌握标准

  • 能从能量守恒解释冲击公式。

  • 能给出降低动应力的措施。

11. 考试模拟与变式训练

数值题：重物重量 W=1 kN 从高度 h=20 mm 落到竖直杆端，杆长 L=1000 mm，面积 A=100 mm^2，弹性模量 E=200000 MPa，许用应力 [sigma]=180 MPa。

  1. 静位移：delta_st=WL/(EA)=1000*1000/(200000*100)=0.05 mm。

变式：h=0 时成为突加载荷，Kd=2。

第13章 疲劳强度与构件寿命估算概述

1. 章节一句话总纲

疲劳问题研究构件在循环应力下的裂纹萌生、扩展和断裂，即使最大应力低于静强度，也必须按应力幅、平均应力和寿命曲线进行校核。

2. 核心问题

问题考试抓手
为什么低于屈服也会断？循环载荷导致裂纹逐步扩展
循环应力怎么描述？sigma_max, sigma_min, sigma_a, sigma_m, R
寿命怎么估？S-N 曲线或疲劳极限
平均应力怎么修正？Goodman 等准则
什么因素最危险？缺口、粗糙表面、尺寸大、残余拉应力、腐蚀

3. 前置知识

  • 内力图和危险截面。
  • sigma=N/A, sigma=My/I, tau=T rho/Ip。
  • 弯扭组合强度理论。
  • 材料强度指标。

4. 概念依赖图

循环外载 -> 受力图 -> 危险截面 -> 循环内力 -> sigma_max/min -> sigma_a/sigma_m -> S-N/Goodman -> 寿命或安全系数

5. 公式血缘表

公式来源适用条件常见误用
sigma_a=(sigma_max-sigma_min)/2应力幅定义循环应力与平均应力混
sigma_m=(sigma_max+sigma_min)/2平均应力定义循环应力符号错
R=sigma_min/sigma_max应力比描述循环类型忽略压应力负号
sigma_a/sigma_-1 + sigma_m/sigma_b <= 1/nGoodman 准则拉伸平均应力修正静强度代替疲劳
sigma_-1,修=k_a k_b k_c k_d / K_f * sigma_-1表面、尺寸、载荷、可靠性、缺口修正的简化乘积实际构件疲劳极限估算用一个 k 淹没所有修正来源

公式血缘审计补充

疲劳公式的前提是循环应力能由内力分析得到，材料 S-N 或疲劳极限数据适用；不适用于一次静载破坏、低周塑性疲劳未给模型、腐蚀/高温无修正数据的问题。sigma_max,sigma_min,sigma_a,sigma_m,R 必须保留拉压符号；应力单位通常为 MPa，寿命为循环次数；典型题入口是轴类弯扭疲劳、缺口疲劳和 S-N 寿命估算。

疲劳修正拆解：k_a 常代表表面质量，k_b 代表尺寸，k_c 代表载荷类型，k_d 代表可靠性或环境，K_f 代表疲劳应力集中。若题目只给总修正系数 k，可用 sigma_-1,修=k sigma_-1；若分别给各项，必须逐项代入，不能把缺口影响漏掉。

6. 典型题型表

题型标准流程
拉压疲劳求循环轴力 -> 求 sigma_max/min -> sigma_a/m -> 校核
弯曲疲劳画弯矩图 -> 找危险截面 -> 求交变弯曲应力
弯扭组合疲劳求 M,T -> 等效应力幅/平均值 -> 疲劳准则
缺口疲劳名义应力 -> 乘疲劳应力集中系数 Kf
寿命估算实际应力幅 -> 查 S-N 曲线 -> 与要求寿命比

7. 易错点清单

  • 把疲劳强度当静强度。
  • 没有先求循环内力。
  • sigma_a 和 sigma_m 算反。
  • 脉动循环误当对称循环。
  • 缺口题不考虑应力集中。

8. 最低掌握标准

  • 能解释疲劳三阶段。
  • 能求 sigma_a, sigma_m, R。
  • 能读简单 S-N 曲线。
  • 能判断疲劳危险因素。

9. 高分掌握标准

  • 能处理弯扭组合疲劳。
  • 能说明疲劳极限修正，并区分表面、尺寸、载荷、可靠性和缺口来源。
  • 能提出改进方案：圆角、抛光、表面强化、降载。

第14章 复合材料力学性能和聚合物粘弹性力学性能

1. 章节一句话总纲

复合材料强调方向性与组分协同，聚合物粘弹性强调时间效应；解题仍要先求内力和应力，再选择对应的材料本构关系。

2. 核心问题

问题考试抓手
复合材料为何不能直接用普通 E？各向异性，性能依赖纤维方向
纵向模量怎么估？等应变规则混合法
横向模量为什么低？主要受基体和界面控制
蠕变是什么？恒定应力下应变随时间增大
松弛是什么？恒定应变下应力随时间降低

3. 前置知识

  • 应力应变关系 sigma=E epsilon, tau=G gamma。
  • 截面法与平均应力。
  • 强度和刚度校核。
  • 基本函数图像和时间变量。

4. 概念依赖图

外载 -> 受力图 -> 截面内力 -> 应力/应变 -> 方向性或时间性本构 -> 变形/强度/长期刚度校核

5. 公式血缘表

公式来源适用条件常见误用
Vf+Vm=1体积分数两相简化复合材料与质量分数混
E1=Vf Ef+Vm Em纵向等应变沿纤维方向拉伸用于横向
1/E2=Vf/Ef+Vm/Em横向等应力近似垂直纤维方向与纵向公式混
epsilon(t)=J(t) sigma蠕变柔量恒定应力漏时间
sigma(t)=E(t) epsilon松弛模量恒定应变把松弛当蠕变
Maxwell 模型 dot epsilon=dot sigma/E+sigma/eta弹簧与黏壶串联应力松弛明显与 Kelvin 模型混
Kelvin 模型 sigma=E epsilon+eta dot epsilon弹簧与黏壶并联蠕变延迟明显误认为能描述瞬时松弛

公式血缘审计补充

复合材料公式前提是体积分数已知、单向连续纤维、简化等应变/等应力假设；不适用于复杂铺层直接等同单向层、界面失效主导或质量分数未换算。粘弹性公式前提是给定加载历史和 J(t) 或 E(t)；不适用于时间条件不明。Vf,Vm,E1,E2,J(t),E(t) 要写清方向和单位；典型题入口是纵向/横向模量、复合杆拉伸、蠕变和应力松弛。

6. 典型题型表

题型标准流程
单向复合材料纵向模量判断沿纤维 -> E1=Vf Ef+Vm Em
横向模量判断垂直纤维 -> 用倒数混合法
复合材料杆拉伸受力图 -> N=P -> sigma=N/A -> 沿纤维用 epsilon=sigma/E1，横向用 epsilon=sigma/E2
蠕变求恒定应力 -> epsilon(t)=J(t)sigma
应力松弛给恒定应变 -> sigma(t)=E(t)epsilon
