# 现实世界中的流体力学：广泛机制版

上一版的问题是：很多例子都属于“高速流导致压强变低”这一类，类型重复。这一版按**不同物理机制**整理，每个例子都写清：

1. 现象是什么；
2. 公式从哪里来；
3. 每个符号是什么意思；
4. 代入一组现实数值会得到多大的结果。

默认常数：

\[
\rho_{\text{air}}\approx 1.2\ \mathrm{kg/m^3},\qquad
\rho_{\text{water}}\approx 1000\ \mathrm{kg/m^3},\qquad
g=9.81\ \mathrm{m/s^2}
\]

## 1. 列车经过时会把人往车边带：动压和压强差

### 现实场景

高铁、地铁、大货车从身边快速经过时，人会感觉被拉向车体或被扰动气流推晃。

### 公式来源

从伯努利方程的能量形式来：

\[
p+\frac12\rho v^2=\text{常数}
\]

如果某处空气速度 \(v\) 变大，那么动能项 \(\frac12\rho v^2\) 变大，静压 \(p\) 就可能变小。现实列车旁边还会有尾流、分离、湍流，所以严格说不是单纯伯努利，但可以用动压估算危险量级：

\[
q=\frac12\rho v^2
\]

\[
F\approx qA
\]

### 符号

| 符号 | 意思 |
|---|---|
| \(q\) | 动压，代表压强差的量级 |
| \(\rho\) | 空气密度 |
| \(v\) | 列车速度或车旁空气速度量级 |
| \(F\) | 人受到的侧向力估算 |
| \(A\) | 人体侧面迎风面积 |

### 数值

列车速度：

\[
v=80\ \mathrm{km/h}=22.2\ \mathrm{m/s}
\]

\[
q=\frac12\times1.2\times22.2^2\approx296\ \mathrm{Pa}
\]

人体侧面积取：

\[
A=0.5\ \mathrm{m^2}
\]

\[
F\approx296\times0.5=148\ \mathrm{N}
\]

若速度变成 \(120\ \mathrm{km/h}\)，则 \(q\approx666\ \mathrm{Pa}\)，力约 \(333\ \mathrm{N}\)。  
重点：速度变大，风险按 \(v^2\) 增长。

## 2. 水龙头流下来的水柱会越来越细：连续性方程

### 现实场景

水从水龙头流出后，越往下水柱越细。

### 公式来源

不可压缩水流满足质量守恒。密度不变时：

\[
Q=Av=\text{常数}
\]

水往下落时重力加速，速度变大。由机械能守恒：

\[
v^2=v_0^2+2gz
\]

速度 \(v\) 变大，而流量 \(Q\) 不变，所以截面积 \(A\) 必须变小，水柱就变细。

### 符号

| 符号 | 意思 |
|---|---|
| \(Q\) | 体积流量 |
| \(A\) | 水柱截面积 |
| \(v\) | 水柱速度 |
| \(v_0\) | 刚出水龙头时的速度 |
| \(z\) | 下落高度 |

### 数值

设刚出水龙头时直径：

\[
d_0=8\ \mathrm{mm}
\]

速度：

\[
v_0=1.0\ \mathrm{m/s}
\]

下落：

\[
z=0.5\ \mathrm{m}
\]

则：

\[
v=\sqrt{1.0^2+2\times9.81\times0.5}
\approx3.29\ \mathrm{m/s}
\]

因为 \(A\propto d^2\)，有：

\[
\frac{d}{d_0}=\sqrt{\frac{v_0}{v}}
\]

\[
d=8\sqrt{\frac{1.0}{3.29}}\approx4.4\ \mathrm{mm}
\]

所以水柱从 \(8\ \mathrm{mm}\) 变到约 \(4.4\ \mathrm{mm}\)。

## 3. 雨滴为什么不会无限加速：阻力和终端速度

### 现实场景

雨滴从高空落下，但不会像石子一样越落越快到恐怖速度。

### 公式来源

雨滴受到重力：

\[
G=mg
\]

也受到空气阻力：

\[
F_D=\frac12\rho C_DAv^2
\]

速度越大，阻力越大。终端速度时：

\[
mg=\frac12\rho C_DAv_t^2
\]

解得：

\[
v_t=\sqrt{\frac{2mg}{\rho C_DA}}
\]

### 符号

| 符号 | 意思 |
|---|---|
| \(m\) | 雨滴质量 |
| \(g\) | 重力加速度 |
| \(F_D\) | 空气阻力 |
| \(\rho\) | 空气密度 |
| \(C_D\) | 阻力系数 |
| \(A\) | 雨滴迎风面积 |
| \(v_t\) | 终端速度 |

### 数值

取直径 \(d=2\ \mathrm{mm}\)，半径 \(r=1\ \mathrm{mm}\)：

\[
m=\rho_{\text{water}}\frac43\pi r^3
\approx4.19\times10^{-6}\ \mathrm{kg}
\]

\[
A=\pi r^2\approx3.14\times10^{-6}\ \mathrm{m^2}
\]

取 \(C_D\approx0.6\)：

\[
v_t\approx6\ \mathrm{m/s}
\]

这就是雨滴不会无限加速的原因。

## 4. 用细吸管喝稠奶茶会很费劲：黏性流动

### 现实场景

同样是吸一口，水很容易，稠奶茶、蜂蜜、糖浆很难。

### 公式来源

细管中的层流黏性流动可以由 Navier-Stokes 方程在圆管内求解得到，结果叫泊肃叶公式：

\[
\Delta p=\frac{8\mu LQ}{\pi r^4}
\]

它说明压强差和黏度成正比，和吸管半径的四次方成反比。

### 符号

| 符号 | 意思 |
|---|---|
| \(\Delta p\) | 嘴需要制造的压强差 |
| \(\mu\) | 动力黏度 |
| \(L\) | 吸管长度 |
| \(Q\) | 体积流量，也就是你吸得多快 |
| \(r\) | 吸管半径 |

### 数值

取：

\[
L=0.20\ \mathrm{m},\quad r=1.5\ \mathrm{mm},\quad Q=5\ \mathrm{mL/s}=5\times10^{-6}\ \mathrm{m^3/s}
\]

水：

\[
\mu=0.001\ \mathrm{Pa\cdot s}
\]

\[
\Delta p\approx0.50\ \mathrm{kPa}
\]

稠奶茶或糖浆，取：

\[
\mu=0.02\ \mathrm{Pa\cdot s}
\]

\[
\Delta p\approx10\ \mathrm{kPa}
\]

所以“难吸”不是错觉。更关键的是：

\[
\Delta p\propto\frac1{r^4}
\]

吸管半径减半，难度会变成 \(16\) 倍。

## 5. 纸巾会自动吸水：表面张力和毛细作用

### 现实场景

纸巾一角碰到水后，水会自己往上爬。

### 公式来源

把纸巾孔隙看成一根很细的毛细管。水被表面张力向上拉：

\[
F_{\sigma}=2\pi r\sigma\cos\theta
\]

水柱重量：

\[
G=\rho g(\pi r^2h)
\]

平衡时：

\[
2\pi r\sigma\cos\theta=\rho g\pi r^2h
\]

解得：

\[
h=\frac{2\sigma\cos\theta}{\rho gr}
\]

### 符号

| 符号 | 意思 |
|---|---|
| \(h\) | 水能爬升的高度 |
| \(\sigma\) | 表面张力系数 |
| \(\theta\) | 接触角 |
| \(\rho\) | 水的密度 |
| \(r\) | 孔隙半径 |

### 数值

水的表面张力：

\[
\sigma\approx0.072\ \mathrm{N/m}
\]

若纸巾孔隙半径：

\[
r=50\ \mu\mathrm{m}=5.0\times10^{-5}\ \mathrm{m}
\]

近似取 \(\cos\theta\approx1\)：

\[
h=\frac{2\times0.072}{1000\times9.81\times5.0\times10^{-5}}
\approx0.29\ \mathrm{m}
\]

这说明纸巾吸水高度达到几十厘米并不夸张。

## 6. 人为什么能浮在水上：静水压和浮力

### 现实场景

人在水中会感觉身体变轻，有些人可以平躺漂浮。

### 公式来源

水越深压强越大：

\[
p=p_0+\rho gh
\]

物体下表面受到的压力比上表面大，压力差合起来形成向上的浮力。积分结果就是阿基米德原理：

\[
F_b=\rho_{\text{water}}gV
\]

### 符号

| 符号 | 意思 |
|---|---|
| \(F_b\) | 浮力 |
| \(\rho_{\text{water}}\) | 水的密度 |
| \(V\) | 被身体排开的水的体积 |
| \(g\) | 重力加速度 |

### 数值

一个人体重：

\[
m=70\ \mathrm{kg}
\]

重力：

\[
G=mg=70\times9.81\approx687\ \mathrm{N}
\]

如果身体体积约：

\[
V=0.071\ \mathrm{m^3}
\]

浮力：

\[
F_b=1000\times9.81\times0.071\approx696\ \mathrm{N}
\]

浮力和重力差不多，所以人可以漂浮。淡水中容易刚好平衡，海水密度更大，所以更容易浮。

## 7. 茶杯或下水口漩涡中心会凹下去：旋转流的径向压强差

### 现实场景

搅动茶水后，液面中心会凹下去；浴缸下水口附近也会出现漩涡。

### 公式来源

旋转流中，流体做圆周运动需要向心力。这个向心力由径向压强梯度提供：

\[
\frac{dp}{dr}=\rho\frac{v_\theta^2}{r}
\]

越靠近中心，压力越低。自由液面上压强近似都是大气压，所以液面高度必须下降来抵消速度造成的压强变化。可用能量估算：

\[
\rho gh\approx\frac12\rho v^2
\]

\[
h\approx\frac{v^2}{2g}
\]

### 符号

| 符号 | 意思 |
|---|---|
| \(p\) | 压强 |
| \(r\) | 到旋涡中心的距离 |
| \(v_\theta\) | 切向速度 |
| \(h\) | 液面下凹高度 |

### 数值

若靠近中心的水速：

\[
v=0.5\ \mathrm{m/s}
\]

\[
h\approx\frac{0.5^2}{2\times9.81}
\approx0.0127\ \mathrm{m}
\]

\[
h\approx1.3\ \mathrm{cm}
\]

所以漩涡中心下凹一两厘米很正常。

## 8. 足球和乒乓球为什么能拐弯：环量和马格努斯效应

### 现实场景

旋转足球会踢出弧线；乒乓球、网球、棒球也会因为旋转改变轨迹。

### 公式来源

旋转球让周围流场出现环量 \(\Gamma\)。二维理想流中，库塔-儒可夫斯基公式为：

\[
L'=\rho U\Gamma
\]

实际球体常用升力系数估算：

\[
F_L=\frac12\rho U^2AC_L
\]

它本质还是：球两侧速度不同，压强不同，于是形成侧向力。

### 符号

| 符号 | 意思 |
|---|---|
| \(L'\) | 单位长度升力 |
| \(\Gamma\) | 环量，表示绕球的旋转流强度 |
| \(U\) | 球相对空气的速度 |
| \(F_L\) | 球受到的侧向力 |
| \(A\) | 球的迎风面积 |
| \(C_L\) | 升力系数，和旋转强弱有关 |

### 数值

足球直径约：

\[
d=0.22\ \mathrm{m}
\]

\[
A=\pi(0.11)^2\approx0.038\ \mathrm{m^2}
\]

球速：

\[
U=25\ \mathrm{m/s}
\]

取 \(C_L=0.15\)：

\[
F_L=\frac12\times1.2\times25^2\times0.038\times0.15
\approx2.1\ \mathrm{N}
\]

足球质量约 \(0.43\ \mathrm{kg}\)，侧向加速度：

\[
a=\frac{F_L}{m}\approx4.9\ \mathrm{m/s^2}
\]

飞行 \(1\ \mathrm{s}\) 时，侧向位移量级：

\[
s\approx\frac12at^2\approx2.5\ \mathrm{m}
\]

所以弧线球的横向偏移可以非常明显。

## 9. 灰尘为什么能在空气中飘很久：低雷诺数黏性阻力

### 现实场景

大颗粒灰尘很快落下，小颗粒灰尘可以在空气中飘很久。

### 公式来源

很小颗粒、低速度时，阻力不是 \(\frac12\rho C_DAv^2\)，而近似为 Stokes 阻力：

\[
F_D=6\pi\mu rv
\]

终端速度时：

\[
\text{重力}-\text{浮力}=\text{黏性阻力}
\]

\[
\frac43\pi r^3(\rho_p-\rho_f)g=6\pi\mu rv
\]

解得：

\[
v=\frac{2(\rho_p-\rho_f)gr^2}{9\mu}
\]

### 符号

| 符号 | 意思 |
|---|---|
| \(r\) | 颗粒半径 |
| \(\rho_p\) | 颗粒密度 |
| \(\rho_f\) | 空气密度 |
| \(\mu\) | 空气动力黏度 |
| \(v\) | 颗粒沉降速度 |

### 数值

取一粒较小灰尘：

\[
r=1\ \mu\mathrm{m}=1.0\times10^{-6}\ \mathrm{m}
\]

\[
\rho_p=2000\ \mathrm{kg/m^3},\quad
\mu_{\text{air}}=1.8\times10^{-5}\ \mathrm{Pa\cdot s}
\]

\[
v=\frac{2(2000-1.2)9.81(10^{-6})^2}{9(1.8\times10^{-5})}
\approx2.4\times10^{-4}\ \mathrm{m/s}
\]

也就是：

\[
v\approx0.24\ \mathrm{mm/s}
\]

它下降 \(1\ \mathrm{m}\) 需要约：

\[
\frac{1}{2.4\times10^{-4}}\approx4200\ \mathrm{s}
\]

约 \(70\) 分钟。小颗粒能飘很久就是这个原因。

## 10. 水槽里薄水膜突然变成厚水圈：水跃和弗劳德数

### 现实场景

水龙头冲到水槽底部后，水先向外形成很薄很快的水膜，到某个圆环位置突然变厚、变慢。

### 公式来源

浅水流中，表面波传播速度约为：

\[
c=\sqrt{gh}
\]

定义弗劳德数：

\[
Fr=\frac{v}{\sqrt{gh}}
\]

如果 \(Fr>1\)，水流比浅水波还快，叫超临界流；如果 \(Fr<1\)，叫亚临界流。薄而快的水膜遇到外部较慢较厚区域时，会发生水跃。

### 符号

| 符号 | 意思 |
|---|---|
| \(Fr\) | 弗劳德数 |
| \(v\) | 水膜平均速度 |
| \(h\) | 水膜厚度 |
| \(c=\sqrt{gh}\) | 浅水波速 |

### 数值

薄水膜区：

\[
h=1\ \mathrm{mm}=0.001\ \mathrm{m},\quad v=1.0\ \mathrm{m/s}
\]

\[
Fr=\frac{1.0}{\sqrt{9.81\times0.001}}
\approx10.1
\]

这是超临界流，速度很快、厚度很薄。

水跃后如果：

\[
h=5\ \mathrm{mm},\quad v=0.2\ \mathrm{m/s}
\]

\[
Fr=\frac{0.2}{\sqrt{9.81\times0.005}}
\approx0.90
\]

这变成亚临界流，水层变厚、速度变慢。水槽里的圆形厚水圈就是这个机制。

## 11. 房间里气味为什么有时很快散开，有时很慢：扩散和对流

### 现实场景

香水、烟味、饭味，在没有风的时候扩散很慢；开窗或开风扇后会快很多。

### 公式来源

只有分子扩散时，扩散时间量级：

\[
t_{\text{diff}}\sim\frac{L^2}{D}
\]

有空气流动时，对流输运时间：

\[
t_{\text{adv}}\sim\frac{L}{U}
\]

### 符号

| 符号 | 意思 |
|---|---|
| \(L\) | 传播距离 |
| \(D\) | 气味分子的扩散系数 |
| \(U\) | 房间空气流速 |
| \(t_{\text{diff}}\) | 纯扩散所需时间 |
| \(t_{\text{adv}}\) | 被气流带走所需时间 |

### 数值

房间尺度：

\[
L=3\ \mathrm{m}
\]

空气中小分子的扩散系数：

\[
D\approx1.0\times10^{-5}\ \mathrm{m^2/s}
\]

纯扩散：

\[
t_{\text{diff}}\sim\frac{3^2}{10^{-5}}=9.0\times10^5\ \mathrm{s}
\]

约 \(10\) 天。

如果有很弱气流：

\[
U=0.1\ \mathrm{m/s}
\]

\[
t_{\text{adv}}\sim\frac{3}{0.1}=30\ \mathrm{s}
\]

所以“开窗通风”不是让分子自己扩散，而是用流动把气味输运出去。
