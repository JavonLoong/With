# 符号词典：湍流近壁与管路题

## 湍流近壁分层

| 符号 | 意思 | 单位 |
|---|---|---|
| \(u\) | 某一点局部速度 | \(\mathrm{m/s}\) |
| \(\bar u\)、\(\bar u_m\) | 截面平均速度 | \(\mathrm{m/s}\) |
| \(u_{\max}\) | 最大速度，圆管中通常在管中心 | \(\mathrm{m/s}\) |
| \(y\) | 离壁面距离 | \(\mathrm{m}\) |
| \(R\) | 管半径 | \(\mathrm{m}\) |
| \(r\) | 离管中心距离 | \(\mathrm{m}\) |
| \(y=R-r\) | 圆管中离壁面距离 | \(\mathrm{m}\) |
| \(\rho\) | 密度 | \(\mathrm{kg/m^3}\) |
| \(\mu\) | 动力粘度 | \(\mathrm{Pa\cdot s}\) |
| \(\nu\) | 运动粘度，\(\nu=\mu/\rho\) | \(\mathrm{m^2/s}\) |
| \(\tau_w\) | 壁面切应力 | \(\mathrm{Pa}\) |
| \(u_\tau\) | 摩擦速度，\(u_\tau=\sqrt{\tau_w/\rho}\) | \(\mathrm{m/s}\) |
| \(u^+\) | 无量纲速度，\(u^+=u/u_\tau\) | 无单位 |
| \(y^+\) | 无量纲壁面距离，\(y^+=yu_\tau/\nu\) | 无单位 |

核心公式：

\[
u_\tau=\sqrt{\frac{\tau_w}{\rho}}
\]

\[
u^+=\frac{u}{u_\tau}
\]

\[
y^+=\frac{yu_\tau}{\nu}
\]

\[
u^+=y^+ \quad (y^+<5)
\]

\[
u^+=2.5\ln y^+ +5.5 \quad (y^+>30)
\]

圆管平均速度公式：

\[
\frac{\bar u}{u_\tau}
=2.5\ln\frac{Ru_\tau}{\nu}+1.75.
\]

圆管最大速度公式：

\[
\frac{u_{\max}}{u_\tau}
=2.5\ln\frac{Ru_\tau}{\nu}+5.5.
\]

## 管路机械能方程

| 符号 | 意思 | 单位 |
|---|---|---|
| \(Q\) | 流量 | \(\mathrm{m^3/s}\) |
| \(A\) | 管道截面积 | \(\mathrm{m^2}\) |
| \(V\) | 管内平均速度，\(V=Q/A\) | \(\mathrm{m/s}\) |
| \(d\) | 管径 | \(\mathrm{m}\) |
| \(L\) | 管长 | \(\mathrm{m}\) |
| \(e\) | 管壁粗糙度 | \(\mathrm{m}\) |
| \(e/d\) | 相对粗糙度 | 无单位 |
| \(Re\) | 雷诺数，\(Re=Vd/\nu\) | 无单位 |
| \(\lambda\) | 沿程阻力系数 | 无单位 |
| \(\zeta\) | 局部阻力系数 | 无单位 |
| \(h_f\) | 沿程水头损失 | \(\mathrm{m}\) |
| \(h_\zeta\) | 局部水头损失 | \(\mathrm{m}\) |
| \(H_p\) | 泵增加的水头 | \(\mathrm{m}\) |
| \(H_T\) | 水轮机取走的水头 | \(\mathrm{m}\) |

沿程损失：

\[
h_f=\lambda\frac{L}{d}\frac{V^2}{2g}.
\]

局部损失：

\[
h_\zeta=\zeta\frac{V^2}{2g}.
\]

雷诺数：

\[
Re=\frac{Vd}{\nu}.
\]

流量速度关系：

\[
V=\frac{Q}{A}=\frac{Q}{\pi d^2/4}.
\]

泵给水加能：

\[
+H_p
\]

水轮机从水中取能：

\[
+H_T
\]

关键区别：

| 题型 | 机器项位置 |
|---|---|
| 泵题 | 左边加 \(H_p\)，表示泵给水增加能量 |
| 水轮机题 | 右边加 \(H_T\)，表示水轮机从水中取走能量 |
