## 六、固体力学/力学类实验高分写法清单

> 这部分是通用的“力学实验满分写法”。如果后续写拉伸、扭转、弯曲、组合梁、电测桥路等固体力学实验，可以直接套这个结构；RLC 报告也可以借用其中的数据处理和误差分析标准。

- [ ] 实验目的不要只抄讲义，要写成“测量量 + 验证规律 + 评价误差”。
  - 普通写法：测量材料弹性模量。
  - 高分写法：通过拉伸实验获得载荷-位移曲线，在线弹性段拟合弹性模量，并比较不同测量方法/试样尺寸对结果的影响。

- [ ] 原理部分要突出核心模型和适用条件。
  - 拉伸实验：胡克定律、应力应变关系、线弹性范围、屈服/强化/断裂阶段。
  - 扭转实验：圆轴扭转公式、剪应力分布、剪切模量、极惯性矩。
  - 弯曲实验：纯弯曲假设、平截面假设、弯曲正应力公式、挠度公式。
  - 组合梁/电测桥路：应变片测量原理、桥路灵敏度、理论应变与实测应变比较。
  - 高分点：说明公式成立条件，避免把理论当成无条件真理。

- [ ] 数据处理必须体现“原始数据 - 派生量 - 拟合结果”的链条。
  - 原始数据表保留载荷、位移、应变、电压等直接读数。
  - 派生数据表写清楚应力、应变、弯矩、剪应力、弹性模量等计算量。
  - 图线优先使用拟合，而不是只用两个点相除。
  - 拟合图要标出方程、相关系数或残差特征。

- [ ] 关键参数优先用线性拟合求，不要只做单点计算。
  - 弹性模量：用线弹性段 `\sigma-\varepsilon` 拟合斜率。
  - 剪切模量：用扭矩-扭转角曲线拟合。
  - 弯曲刚度：用载荷-挠度或弯矩-应变曲线拟合。
  - 高分点：说明为什么选取这一段数据作为拟合区间。

- [ ] 图表要像“论文图”，不是截图堆叠。
  - 图题写清楚实验对象、变量关系和单位。
  - 坐标轴必须有物理量和单位。
  - 数据点和拟合线要区分。
  - 如果有理论值，图中可以加理论线或误差带。
  - 图下正文必须解释图像趋势，而不是只说“如图所示”。

- [ ] 误差分析要定量化。
  - 至少给出相对误差：`|实验值-理论值|/理论值 \times 100%`。
  - 能做不确定度时，写出主要不确定度来源和传播关系。
  - 对力学实验常见误差要写具体：试样尺寸测量、夹具间隙、初始零点、加载不均、应变片粘贴方向、材料非理想均匀、读数滞后。
  - 高分点：说明误差会让结果偏大还是偏小。

- [ ] 讨论部分要有“反常点处理”。
  - 如果某几个数据点偏离拟合线，要说明可能原因。
  - 不要随便删除异常点；若删除，要给出理由。
  - 可以补一句“保留异常点时拟合结果为……，剔除后为……”，体现你知道数据处理的边界。

- [ ] 结论部分要写成定量条目。
  - 例如：测得弹性模量 `E = ... GPa`，相对误差 `...%`。
  - 验证了线弹性阶段 `\sigma` 与 `\varepsilon` 近似线性，拟合相关系数 `r = ...`。
  - 主要误差来自试样直径测量和夹具滑移，改进方案是多点测径、预载消隙、增加重复测量。
