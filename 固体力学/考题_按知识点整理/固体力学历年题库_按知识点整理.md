## 90+ 刷题顺序

- **T1 内力图、剪力图、弯矩图**（A级，29 题/片段）：受力图 -> 支反力 -> 截面法 -> 分段写 V/M -> 画图并查端点、跳跃、面积关系。
- **T2 梁弯曲应力、截面性质、强度校核**（A级，55 题/片段）：先由弯矩图找危险截面，再算 I/W/Q，最后写 sigma=My/I 或 tau=VQ/(Ib) 并校核。
- **T3 梁位移、刚度、能量法**（A级，23 题/片段）：先写 M(x)，再选积分法、叠加法或单位载荷法；边界条件和单位载荷方向是得分核心。
- **T4 应力状态、莫尔圆、强度理论、组合变形**（A级，20 题/片段）：先求危险点的 sigma/tau，再求主应力或最大剪应力，最后选择第三/第四强度理论。
- **T5 轴向拉压、变形、静不定、温度应力**（B+级，6 题/片段）：轴力图 -> sigma=N/A -> Delta l=sum NL/EA；静不定题补平衡、协调、本构三件套。
- **T6 扭转、连接件剪切/挤压**（B级，23 题/片段）：扭矩图 -> tau=T rho/Ip -> phi=TL/GIp；连接件先找剪切面和挤压面积。
- **T7 压杆稳定**（B级，3 题/片段）：端部约束 -> 有效长度 -> 弱轴惯性矩 -> Pcr=pi^2EI/(mu L)^2 -> 稳定安全系数。
- **T8 动载、疲劳、复合材料、粘弹性**（C级，1 题/片段）：低频但要会识别：动载看动荷系数，疲劳看应力幅/平均应力，复合/粘弹看方向性和时间效应。
- **T9 基本概念、单位、假设、边界条件**（C级，29 题/片段）：最后扫雷：基本假设、单位换算、边界条件、强度/刚度/稳定差异。
