# 管路机械能方程：通用流程

第2题和第3题都属于管路机械能方程题，不属于第1题那种 \(u^+,y^+\) 的近壁湍流分层题。

## 统一方程

压力形式：

\[
p_A+\rho gz_A+\frac12\rho V_A^2+\Delta p_{\text{pump}}
=p_B+\rho gz_B+\frac12\rho V_B^2+\Delta p_f+\sum\Delta p_\zeta.
\]

水头形式：

\[
\frac{p_A}{\rho g}+z_A+\frac{V_A^2}{2g}+H_p
=
\frac{p_B}{\rho g}+z_B+\frac{V_B^2}{2g}+h_f+\sum h_\zeta.
\]

如果是水轮机，不是加能，而是取能：

\[
\frac{p_A}{\rho g}+z_A+\frac{V_A^2}{2g}
=
\frac{p_B}{\rho g}+z_B+\frac{V_B^2}{2g}+h_f+h_T.
\]

## 每一项什么意思

| 项 | 意思 |
|---|---|
| \(p/\rho g\) | 压强水头 |
| \(z\) | 位置水头 |
| \(V^2/(2g)\) | 速度水头 |
| \(H_p\) | 泵给流体增加的水头 |
| \(h_T\) | 水轮机从流体取走的水头 |
| \(h_f\) | 沿程损失 |
| \(h_\zeta\) | 局部损失 |

## 损失公式

沿程损失：

\[
h_f=\lambda\frac{L}{d}\frac{V^2}{2g}.
\]

局部损失：

\[
h_\zeta=\zeta\frac{V^2}{2g}.
\]

多个局部件：

\[
\sum h_\zeta=(\zeta_1+\zeta_2+\cdots)\frac{V^2}{2g}.
\]

## 阻力系数 \(\lambda\) 怎么来

先算速度：

\[
V=\frac{Q}{A}=\frac{Q}{\pi d^2/4}.
\]

再算雷诺数：

\[
Re=\frac{Vd}{\nu}.
\]

再算相对粗糙度：

\[
\frac{e}{d}.
\]

用 \(Re\) 和 \(e/d\) 查莫迪图得到 \(\lambda\)。

## 做题模板

1. 选两个截面 \(A\rightarrow B\)。
2. 判断有没有泵或水轮机。
3. 写机械能方程。
4. 算管内速度 \(V=Q/A\)。
5. 算 \(Re\)。
6. 算 \(e/d\)，查 \(\lambda\)。
7. 算 \(h_f\) 和 \(\sum h_\zeta\)。
8. 代回方程求未知量。

关键句：

\[
\boxed{\text{管路题不是先找公式，而是先选 A、B 两点。}}
\]
