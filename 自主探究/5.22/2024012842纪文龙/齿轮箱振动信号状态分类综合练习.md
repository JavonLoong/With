# 综合练习题：齿轮箱振动信号状态识别入门

## 一、题目背景

齿轮箱是机械传动系统中的关键部件。不同健康状态下，采集到的振动数据会呈现不同的数值特征。现提供齿轮箱振动信号数据文件 `data_train.npz` 和 `data_test.npz`，以及一套基础参考程序。

本练习的重点不是讲解机器学习算法原理，而是借助一个简化的状态识别场景，综合复习 Python 基础语法、NumPy、pandas、Matplotlib 和文件组织方式。`sklearn` 可以作为现成工具练习调用，但其中模型方法的数学原理不作为本课程重点。

请在理解程序结构的基础上，完成数据读取、数组检查、基础摘要统计、简单可视化、模型调用、结果输出和报告解读。

## 二、练习目标

本题目综合复习以下知识点：

- Python 基本语法、变量、表达式和格式化输出
- 列表、字典、元组等常用数据结构
- 流程控制
- 函数封装与参数传递
- 类与对象的基本组织方式
- 文件路径、文件 I/O 与异常处理
- NumPy 数据读取、数组形状、数组拼接和基础统计
- pandas 表格整理、CSV 读取与结果汇总
- Matplotlib 基础绘图与图片保存
- `sklearn` 基础接口调用：`fit()`、`predict()`，不展开算法原理
