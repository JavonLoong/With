# RLC 报告模板要求对照

## 已提取的模板要求

- 标题信息：实验名称、姓名、班级、学号、日期。
- 实验目的：简要、实事求是，不浮夸。
- 实验原理：不能照抄讲义，应总结凝练；一般不超过 1 页 A4；公式按顺序编号并可在正文引用。
- 字体与符号：中文宋体，英文 Times New Roman；公式和物理符号注意斜体；单位规范。
- 实验仪器：简要总结；如放图，图片清晰并有图注。
- 实验任务及步骤：精炼描述自己所做实验，不照抄讲义。
- 数据处理：表格有编号、表头、单位；数据居中；图有坐标、单位、图例；拟合给出方程、图像、相关系数。
- 不确定度/误差：复杂推导可放附录；不确定度有效位数 1-2 位。
- 实验小结：严谨求实，可总结、反思、建议。
- 思考题：按个人理解回答，避免照抄。
- 参考文献：借鉴讲义、资料、网站时列出文献，按出现顺序编号。
- 附录：原始数据一定附后，便于查证。

## 对现有 RLC 报告的重整动作

- [x] 保留并规范标题信息、实验目的、实验原理、仪器、步骤、数据处理、误差分析、小结、思考题。
- [x] 确认表格均有编号、表头和单位，图像含坐标轴、单位和图例。
- [x] 确认阻尼拟合给出拟合方程、拟合图和相关系数。
- [x] 补充 `参考文献`。
- [x] 补充 `附录：原始数据记录摘录`。
- [x] 保持欧姆符号修复后的写法：`\mathrm{k}\Omega`，避免 `k□`。
- [x] 显式设置中文宋体、英文 Times New Roman，并为中文粗体指定黑体，减少字体替代风险。
