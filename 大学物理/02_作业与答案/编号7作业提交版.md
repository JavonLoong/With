## 7.2

已知在 `x = 0.1 m` 处：

```text
y(0.1,t)=0.05 sin(1.0-4.0t)
```

由题可读出

```text
A = 0.05 m,   omega = 4.0 rad/s
```

波速 `u = 0.8 m/s`，所以

```text
k = omega/u = 4.0/0.8 = 5.0 rad/m
```

因题给相位形式为 `sin(1.0-4.0t)`，可设波函数为

```text
y(x,t)=0.05 sin(5.0x-4.0t+C)
```

代入 `x = 0.1 m`：

```text
0.05 sin(0.5-4.0t+C)=0.05 sin(1.0-4.0t)
```

故

```text
0.5+C=1.0,   C=0.5
```

所以波函数为

```text
y(x,t)=0.05 sin(5.0x-4.0t+0.5)  m
```

该形式为 `sin(kx-omega t+C)`，表示波沿 `+x` 方向传播。

## 7.5

由图读出：

```text
A = 0.04 m,   lambda = 0.4 m,   u = 0.08 m/s
```

因此

```text
k = 2pi/lambda = 2pi/0.4 = 5pi rad/m
omega = ku = 5pi*0.08 = 0.4pi rad/s
T = 2pi/omega = 5 s
```

波沿 `+x` 方向传播，取

```text
y=A cos(omega t-kx+phi0)
```

由图知 `t=0, x=0` 时 `y=0`，故

```text
cos phi0 = 0
```

又由波形向右传播可知原点质点此后向 `+y` 方向运动，所以

```text
partial y/partial t |_(0,0) > 0
```

而

```text
partial y/partial t = -A omega sin(omega t-kx+phi0)
```

于是

```text
-A omega sin phi0 > 0,   sin phi0 < 0
```

故取

```text
phi0 = -pi/2
```

波函数为

```text
y = 0.04 cos(0.4pi t-5pi x-pi/2)
```

即

```text
y = 0.04 sin(0.4pi t-5pi x)  m
```

```text
uT/8 = 0.08*(5/8)=0.05 m = lambda/8
```

因此 `t=T/8` 的波形为原 `t=0` 波形向右平移 `0.05 m`。

```text
y(x,T/8)=0.04 sin(pi/4-5pi x)
```

## 7.6


### （1）

波峰条件为

```text
cos pi(4t+2x)=1
```

即

```text
pi(4t+2x)=2npi
```

所以

```text
4t+2x=2n
x=n-2t
```

当 `t=4.2 s` 时

```text
x=n-8.4  m,   n=0, +/-1, +/-2, ...
```

距离原点最近的波峰比较：

```text
n=8: x=-0.4 m
n=9: x=0.6 m
```

故距原点最近的波峰在

```text
x=-0.4 m
```

该波峰对应 `n=8`。令其通过原点 `x=0`：

```text
0=n-2t
t=n/2=8/2=4.0 s
```

所以该波峰在 `t=4.0 s` 通过原点。

## 7.10

已知：

```text
v = 5.1*10^3 m/s
A = 1.0*10^-9 m
nu = 1.0*10^3 Hz
rho = 7.9*10^3 kg/m^3
S = 15 cm^2 = 15*10^-4 m^2
```

角频率

```text
omega = 2pi nu = 2pi*10^3 rad/s
```

声波强度

```text
I = (1/2) rho v omega^2 A^2
```

代入：

```text
I = (1/2)*(7.9*10^3)*(5.1*10^3)*(2pi*10^3)^2*(10^-9)^2
  ≈ 7.95*10^-4 W/m^2
```

通过钢轨输送的功率

```text
P = IS
  = 7.95*10^-4 * 15*10^-4
  ≈ 1.19*10^-6 W
```

所以

```text
I ≈ 7.95*10^-4 W/m^2
P ≈ 1.2*10^-6 W
```

## 7.14

设平面简谐波沿 `+x` 方向传播，振幅为 `A`，频率为 `nu`，波速为 `u`。有

```text
omega = 2pi nu
k = 2pi/lambda = omega/u
```

`t=0, x=0` 时，质元在平衡位置且向正方向运动。取入射波

```text
y1 = A cos(omega t-kx+phi0)
```

由 `y1(0,0)=0` 得

```text
cos phi0=0
```

又

```text
partial y1/partial t = -A omega sin(omega t-kx+phi0)
```

在 `t=0, x=0` 有 `partial y1/partial t>0`，故

```text
sin phi0<0
```

因此

```text
phi0=-pi/2
```

入射波为

```text
y1=A cos(omega t-kx-pi/2)=A sin(omega t-kx)
```

即

```text
y1=A sin(2pi nu t-2pi x/lambda)
```

设反射面为波密媒质反射面，即固定端反射，反射点为波节。反射面在

```text
x0 = 3lambda/4
```

要求

```text
y1(x0,t)+y2(x0,t)=0
```

反射波沿 `-x` 方向传播，可设

```text
y2=A sin(omega t+kx+phi)
```

在 `x0=3lambda/4` 处：

```text
y1(x0,t)=A sin(omega t-3pi/2)=A cos(omega t)
```

要使合位移为零，需要

```text
y2(x0,t)=-A cos(omega t)
```

取

```text
y2=A sin(omega t+kx)
```

则

```text
y2(x0,t)=A sin(omega t+3pi/2)=-A cos(omega t)
```

满足固定端边界条件。

所以反射波为

```text
y2=A sin(omega t+kx)
```

合成驻波为

```text
y = y1+y2
  = A sin(omega t-kx)+A sin(omega t+kx)
  = 2A sin(omega t) cos(kx)
```

驻波波节满足

```text
cos(kx)=0
```

即

```text
kx = (2n+1)pi/2
x = (2n+1)lambda/4
```

在 `0 <= x <= 3lambda/4` 范围内，波节为

```text
x = lambda/4,  3lambda/4
```

## 7.15

已知：

```text
d = 2.00 mm = 2.00*10^-3 m
v = 5.74*10^3 m/s
```

石英片两表面为自由面，均为波腹。基频时两个相邻波腹之间距离为半个波长：

```text
d = lambda/2
```

所以

```text
lambda = 2d = 4.00*10^-3 m
```

基频

```text
f = v/lambda
  = 5.74*10^3/(4.00*10^-3)
  = 1.435*10^6 Hz
```

故

```text
f ≈ 1.44 MHz
```

## 7.22

已知：

```text
f0 = 1.8*10^4 Hz
Delta f = 220 Hz
v = 1.54*10^3 m/s
```

驱逐舰静止，潜艇向驱逐舰运动，设潜艇速度为 `vs`。

第一步：潜艇作为运动接收者接收声波：

```text
f1 = f0 (v+vs)/v
```

第二步：潜艇反射声波，相当于运动声源，驱逐舰接收：

```text
f2 = f1 v/(v-vs)
```

因此

```text
f2 = f0 (v+vs)/(v-vs)
```

频差为

```text
Delta f = f2-f0
        = f0[(v+vs)/(v-vs)-1]
        = f0 * 2vs/(v-vs)
```

代入：

```text
220 = 1.8*10^4 * 2vs/(1540-vs)
```

解得

```text
220(1540-vs)=36000vs
338800=36220vs
vs=338800/36220≈9.35 m/s
```

所以潜艇速度为

```text
vs≈9.4 m/s
```
