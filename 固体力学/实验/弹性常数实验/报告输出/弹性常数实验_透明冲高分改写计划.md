# 弹性常数实验报告透明冲高分改写计划

更新时间：2026-04-26

## 目标

在不虚构本人参加现场实验的前提下，把现有《弹性常数测定实验报告》改成可提交的高质量版本。报告要明确说明：当前数据来自课程资料/同组原始记录照片，本文完成的是数据整理、计算、拟合、误差分析与补做准备；后续补做实验后，将用本人实测原始数据替换附录和计算结果。

目标分数定位：报告文本与数据分析质量按 9.5-10 分标准打磨；是否因未现场实验扣分，取决于课程规则和教师裁量，报告中不做虚假承诺。

## 设计原则

- 透明：不写“本人已完成现场测量”之类不真实表述。
- 高分：数据链完整，原始读数、派生量、拟合、相对误差、结论互相可追溯。
- 可补做：给出补做计划，明确补做时需要采集的通道、载荷点、重复性判据和替换位置。
- 不滥用兜底：不把所有问题归为“仪器误差”，误差分析必须针对本实验数据。
- 不跑题：保持固体力学实验报告格式，不引入 RLC 或其它实验内容。

## 必改项

- [ ] 新增“数据来源与提交说明”：说明未参加本次现场测量，当前采用课程资料/同组原始记录照片完成计算分析，补做后替换。
- [ ] 新增“补做计划”：列出载荷点、通道、清零、重复加载、组桥测量、数据合格标准。
- [ ] 新增“核心结果汇总表”：列出 \(E\)、\(\nu\)、拟合 \(R^2\)、中/下截面差异、半桥/对臂桥臂系数及相对误差。
- [ ] 强化“材料判定说明”：解释按铝合金名义截面计算得到 \(E\approx 73\ \mathrm{GPa}\)，与铝合金典型范围一致。
- [ ] 强化定量误差分析：计算左右对称末载差异、两次加载重复性、中/下截面差异、桥臂系数偏差。
- [ ] 调整实验步骤措辞：避免写成“我已现场操作”，改为“现场实验应按以下流程进行；本报告数据处理按该流程对应数据完成”。
- [ ] 结论改成“基于现有记录数据”的结论，避免冒充本人实测结论。
- [ ] 保持参考文献、附录原始照片、公式编号、表格编号和单位规范。

## 子 Agent 分工

- 执行子 agent：使用 `gpt-5.5`，reasoning `xhigh`。只负责修改正式 LaTeX 报告和必要的配套文件，不得调用孙 agent。
- 审核子 agent：使用 `gpt-5.4`，reasoning `xhigh`。只负责复核内容质量、诚信边界、数据一致性和排版风险，不得调用孙 agent。
- 主调度：负责维护本计划文档、派发任务、编译检查、纠偏整合和最终交付。

## 验收标准

- [ ] PDF 能用 XeLaTeX 编译通过。
- [ ] 日志无 `Undefined references`、`Missing character`、`Error`。
- [ ] 报告中没有“本人现场测得”“本次本人操作”等不真实表述。
- [ ] 数据来源说明清楚，但语气不自我贬低。
- [ ] 数据分析至少包含 4 个定量评价：拟合优度、中/下截面差异、桥臂系数偏差、重复性/对称性指标。
- [ ] 附录保留原始记录照片，正文说明后续补做替换方式。
